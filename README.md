# Fumbling Around

This is a general monorepo for small hobby projects.

## Projects

### DocSortBot
A CLI tool for automating the management of financial documents. It includes content-aware year extraction and persistent background operation on macOS.

### CribbageBots
A Cribbage tournament engine that pits different bots against each other. It includes a Python backend (FastAPI) and a modern web frontend (Vite/React) for visualizing games and tournament results.
