# CribbageBots Backend

This is the FastAPI backend and game engine for the Cribbage Tournament Engine.

## Features
- Optimized Cribbage engine (1,000 game bulk simulations supported)
- FastAPI tournament API & WebSocket endpoints
- Real-time Human vs Bot interactive gameplay
- RemoteBot architectural support for BYO-Bot
- Greedy and Random AI bots built-in
